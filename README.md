# DOM Lab Repository

A hands-on JavaScript repository designed to master the **Document Object Model (DOM)** through practical exercises and mini-projects.

---

## 📌 About

This repository focuses on building a strong foundation in DOM manipulation by working on real-world scenarios. The goal is to learn by doing—understanding how JavaScript interacts with the browser to create dynamic and interactive web pages.

---

## Objectives

- Understand DOM structure and behavior  
- Practice selecting and modifying elements  
- Handle events efficiently  
- Work with forms and validations  
- Build interactive UI components using vanilla JavaScript  

---

## How to Use

1. Clone the repository:
```bash
git clone https://github.com/tejaskhurdwork/DOM.git
```

2. Navigate through folders and practice each concept

3. You can experiment by modifying the code and building your own variations
